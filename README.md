## ğŸ³ Installation

The easiest way to run AuraControl is using **Docker Compose**.  This will set up the Frontend, Backend, and Database automatically.

### Step 1: Clone the Repository

```bash
git clone https://github.com/justhoadoan/AuraControl-Spa-Management-System.git
cd AuraControl-Spa-Management-System
```

### Step 2: Configure Environment *(Optional)*

If you need to customize database passwords or ports, edit the `docker-compose.yml` file.

> ğŸ’¡ **Note:** By default, the backend runs on port `8081`

### Step 3: Build and Run

Execute the following command in the root directory:

```bash
docker-compose up -d --build
```
